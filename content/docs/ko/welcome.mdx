# 환영합니다

AI Coding Stack 문서에 오신 것을 환영합니다. AI Coding Stack은 AI 코딩 생태계를 위한 종합 디렉토리 및 메타데이터 저장소입니다.

## AI Coding Stack이란?

AI Coding Stack은 두 가지 주요 목적을 제공합니다:

1. AI 코딩 도구를 발견하고 비교하기 위한 [aicodingstack.io](https://aicodingstack.io)의 공개 웹사이트
2. 코딩 도구, 모델 및 플랫폼에 대한 큐레이션된 정보를 포함하는 커뮤니티 유지 관리 메타데이터 저장소

## 주요 기능

### 포괄적인 커버리지

여러 범주에 걸친 광범위한 AI 코딩 도구 탐색:
- **IDEs** - AI 통합이 있는 통합 개발 환경
- **CLIs** - 명령줄 AI 코딩 어시스턴트
- **Models** - 코딩 작업에 최적화된 대규모 언어 모델
- **Providers** - LLM API 제공업체 및 플랫폼

### 커뮤니티 주도

모든 메타데이터는 JSON 매니페스트 파일로 유지 관리되어 커뮤니티가 다양한 도구에 대한 정보를 기여, 업데이트 및 개선하기 쉽습니다.

### 쉬운 발견

직관적인 웹사이트 인터페이스를 통해 워크플로우에 적합한 AI 코딩 도구를 탐색, 비교 및 발견합니다.
