# 欢迎

欢迎来到 AI Coding Stack 文档。AI Coding Stack 是一个全面的 AI 编码生态系统目录和元数据仓库。

## 什么是 AI Coding Stack？

AI Coding Stack 有两个主要用途：

1. **公共网站** - 通过 [aicodingstack.io](https://aicodingstack.io) 发现和比较 AI 编码工具
2. **社区维护的元数据仓库** - 包含关于编码工具、模型和平台的精选信息

## 核心功能

### 全面覆盖
探索多个类别的 AI 编码工具：
- **终端** - 具有 AI 功能的现代终端模拟器
- **IDE** - 集成 AI 功能的集成开发环境
- **命令行工具** - AI 编码命令行助手
- **模型** - 专为编码任务优化的大语言模型
- **提供商** - LLM API 提供商和平台

### 社区驱动
所有元数据都保存在 JSON 清单文件中，便于社区贡献、更新和改进各种工具的信息。

### 轻松发现
通过直观的网站界面浏览、比较和发现适合你工作流程的 AI 编码工具。
