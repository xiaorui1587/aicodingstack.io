# What is AI Coding Stack and how does it help developers?

AI Coding Stack is a comprehensive directory and metadata repository for the AI coding ecosystem. It provides a centralized platform to discover, compare, and explore AI-powered development tools including IDEs like VS Code and Cursor, command-line assistants like Claude Code, LLM models, and API providers. Instead of searching across scattered resources, AI Coding Stack gives you detailed specifications, pricing, and feature comparisons all in one place.

# How do I use AI Coding Stack?

AI Coding Stack is a website-based directory that you can browse directly at aicodingstack.io. No installation required! Simply:

- **Explore Categories**: Browse through IDEs, CLIs, Models, and Providers
- **Compare Tools**: Use the comparison feature to view specs and pricing side-by-side
- **Read Documentation**: Access setup guides and best practices for each tool
- **Stay Updated**: Check the latest versions and specifications for all tools

The platform is continuously updated with new tools and information as they become available.

# Which AI coding tools and LLM providers are listed in AI Coding Stack?

AI Coding Stack features a comprehensive catalog of AI coding tools and services:

- **IDEs**: Visual Studio Code, Cursor, TRAE, and other AI-enhanced code editors
- **CLIs**: Claude Code, Codex, and other command-line AI assistants
- **Extensions**: GitHub Copilot, Cline, Continue, Roo Code, and other AI code assistant plugins
- **Models**: DeepSeek, Kimi K2, GLM, Qwen, GPT, Claude and other leading AI models
- **Providers**: OpenAI, Anthropic, DeepSeek, Moonshot, OpenRouter, SiliconFlow, and more

The directory is continuously updated as new tools and providers emerge.

# Is AI Coding Stack open source? Can I contribute?

Yes, AI Coding Stack is fully open source and released under the Apache 2.0 license. The project welcomes contributions from the community! You can contribute by:

- Adding or updating tool metadata in the manifests directory (ides.json, clis.json, models.json, providers.json)
- Reporting inaccuracies or outdated information
- Suggesting new tools to be added
- Improving documentation and guides
- Submitting pull requests

Visit the [GitHub repository](https://github.com/aicodingstack/aicodingstack) to get started with contributing to the AI coding ecosystem.

# How does AI Coding Stack help me choose the right tools?

AI Coding Stack provides detailed comparison features and specifications to help you make informed decisions:

- **Side-by-Side Comparison**: Compare multiple tools across key features, pricing, and capabilities
- **Detailed Specifications**: View context windows, token limits, pricing tiers, and platform support
- **Latest Version Tracking**: Stay up-to-date with the newest releases and features
- **Community-Maintained**: Benefit from curated information reviewed by developers
- **Comprehensive Metadata**: Access official documentation, download links, and community resources

All information is organized to help you quickly identify which tools best fit your development workflow.
