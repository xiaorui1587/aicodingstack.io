# AI Coding Stack이란 무엇이며 개발자에게 어떻게 도움이 되나요?

AI Coding Stack은 AI 코딩 생태계를 위한 종합 디렉토리 및 메타데이터 저장소입니다. VS Code 및 Cursor와 같은 IDE, Claude Code와 같은 명령줄 어시스턴트, LLM 모델 및 API 제공업체를 포함한 AI 기반 개발 도구를 발견, 비교 및 탐색할 수 있는 중앙 집중식 플랫폼을 제공합니다. 흩어진 리소스를 검색하는 대신, AI Coding Stack은 한 곳에서 상세한 사양, 가격 및 기능 비교를 제공합니다.

# AI Coding Stack을 어떻게 사용하나요?

AI Coding Stack은 aicodingstack.io에서 직접 탐색할 수 있는 웹사이트 기반 디렉토리입니다. 설치가 필요 없습니다! 간단히:

- **카테고리 탐색**: IDEs, CLIs, Models 및 Providers를 통해 탐색
- **도구 비교**: 비교 기능을 사용하여 사양과 가격을 나란히 보기
- **문서 읽기**: 각 도구에 대한 설정 가이드 및 모범 사례 액세스
- **최신 정보 유지**: 모든 도구의 최신 버전 및 사양 확인

플랫폼은 사용 가능해지면 새로운 도구 및 정보로 지속적으로 업데이트됩니다.

# AI Coding Stack에 어떤 AI 코딩 도구 및 LLM 제공업체가 나열되어 있나요?

AI Coding Stack은 AI 코딩 도구 및 서비스의 종합 카탈로그를 제공합니다:

- **IDEs**: Visual Studio Code, Cursor, TRAE 및 기타 AI 강화 코드 편집기
- **CLIs**: Claude Code, Codex 및 기타 명령줄 AI 어시스턴트
- **Extensions**: GitHub Copilot, Cline, Continue, Roo Code 및 기타 AI 코드 어시스턴트 플러그인
- **Models**: DeepSeek, Kimi K2, GLM, Qwen, GPT, Claude 및 기타 주요 AI 모델
- **Providers**: OpenAI, Anthropic, DeepSeek, Moonshot, OpenRouter, SiliconFlow 등

디렉토리는 새로운 도구 및 제공업체가 등장함에 따라 지속적으로 업데이트됩니다.

# AI Coding Stack은 오픈 소스인가요? 기여할 수 있나요?

네, AI Coding Stack은 Apache 2.0 라이선스 하에 완전히 오픈 소스입니다. 프로젝트는 커뮤니티의 기여를 환영합니다! 다음을 통해 기여할 수 있습니다:

- 매니페스트 디렉토리(ides.json, clis.json, models.json, providers.json)에 도구 메타데이터 추가 또는 업데이트
- 부정확하거나 오래된 정보 보고
- 추가할 새 도구 제안
- 문서 및 가이드 개선
- Pull request 제출

[GitHub 저장소](https://github.com/aicodingstack/aicodingstack.io)를 방문하여 AI 코딩 생태계에 기여를 시작하세요.

# AI Coding Stack이 올바른 도구를 선택하는 데 어떻게 도움이 되나요?

AI Coding Stack은 정보에 입각한 결정을 내리는 데 도움이 되는 상세한 비교 기능 및 사양을 제공합니다:

- **나란히 비교**: 주요 기능, 가격 및 기능에 걸쳐 여러 도구 비교
- **상세한 사양**: 컨텍스트 창, 토큰 제한, 가격 등급 및 플랫폼 지원 보기
- **최신 버전 추적**: 최신 릴리스 및 기능으로 최신 상태 유지
- **커뮤니티 유지 관리**: 개발자가 검토한 큐레이션된 정보 활용
- **종합 메타데이터**: 공식 문서, 다운로드 링크 및 커뮤니티 리소스 액세스

모든 정보는 개발 워크플로우에 가장 적합한 도구를 빠르게 식별하는 데 도움이 되도록 구성되어 있습니다.
